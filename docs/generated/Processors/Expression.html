Expression Evaluator

The Expression Evaluator performs calculations on incoming data on a row-by-row basis. It writes the results of the calculations to the expression output field.

To create an expression, you define the name of the field to receive the results of the expression and the expression that you want to use. Use the expression language to define the calculations that you want to use.

You can define constants that you can use in any expression in the processor. Define a constant when you have a constant value that you use in more than one expression or that you want to change easily.

Expression Output Field

When you configure an expression, the Expression Evaluator writes the results of the expression to a field. You can use an existing field or create a new field.

When you use an existing field, the Expression Evaluator replaces the incoming value with the new value. When you use a new field, the Expression Evaluator adds the new field to the record and passes the value of the expression.

Configuring an Expression Evaluator

Configure an Expression Evaluator to perform calculations on records on a row-by-row basis.

  1. In the Properties panel, on the General tab, configure the following properties.
    General Property Description
    Name Stage name.
    Description Optional description.
    Required Fields Fields that must include data to be passed to the stage.
    Tip: You might include fields that the stage uses.
    On Record Error Error record handling for the stage:
    • Discard - Discards the record.
    • Send to Error - Sends the record to the pipeline for error handling.
    • Stop Pipeline - Stops the pipeline.
  2. To configure an expression, click the Expressions tab and configure the following properties:
    Expression Evaluator Property Description
    Expressions

    Performs calculations within a row and passes the values to an output field. Click the Add icon to add additional expressions.

    • Output Field - The field to pass the results of the expression to the next stage. Enter the name of a new or existing field as follows: /FieldName.

      The Expression Evaluator replaces values in existing field.

    • Expression - Expression to evaluate.
    Constants Represents a constant in expressions. Define a constant to easily reuse it in the stage or to easily update the constant value.

    To define a constant, click the Add icon, and then enter a Name and Value for the constant.
