Installing Additional Stage Libraries

The Data Collector includes a wide range of stages for pipeline development, but you might find a need for additional functionality. You can install additional stage libraries to provide access to additional stages.

Installing a Stage Library (Download-Untar)

You can install a stage library by downloading the library to the appropriate directory and untarring it.

  1. Download the stage library to the parent directory of the Data Collector installation directory.
    For example, if the Data Collector is installed in the /opt directory, you would download the stage library to /opt.
  2. Untar the file.
    For example:
    tar xvf streamsets-datacollector-cdh_5_3_0-lib-1.0.0.tar.gz
  3. Stop and restart the Data Collector.

Installing a Stage Library (Untar-Copy)

You can install a stage library by untarring the library and copying files.

  1. Download the stage library and untar the file.
    For example:
    tar xvf streamsets-datacollector-cdh_5_3_0-lib-1.0.0.tar.gz
  2. Copy the contents of the stage-libs directory to the Data Collector stage-libs directory.
    For example:
    cp -r \
streamsets-datacollector-1.0.0/stage-libs/streamsets-datacollector-cdh5_3_0-lib/ \
/tmp/streamsets-datacollector-1.0.0/stage-libs/
  3. Restart the Data Collector.
