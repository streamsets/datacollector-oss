Upgrade

You can upgrade a previous version of Data Collector to a new version. You can upgrade an installation from the tarball, an installation from the RPM package, or an installation with Cloudera Manager.

Important: The minimum requirements for Data Collector can change with each version. Before you upgrade to a new version, verify that the machine meets the latest minimum requirements as described in Installation Requirements.
