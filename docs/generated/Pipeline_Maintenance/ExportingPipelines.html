Exporting Pipelines

Export pipelines to create backups or to use the pipelines with another Data Collector.

To export pipelines for use in Dataflow Performance Manager (DPM), see Exporting Pipelines for DPM.

When you export a pipeline, Data Collector generates a JSON file named after the pipeline, as follows: <pipeline name>.json.

  1. From the Home page, select pipelines in the list, click the More icon, and then click Export.
  2. Or, to export a pipeline in the pipeline canvas, click the More icon for the pipeline and then click Export.
    Data Collector generates the JSON files and saves them to your default downloads directory.
