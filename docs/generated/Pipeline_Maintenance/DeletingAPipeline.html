Deleting a Pipeline

You can delete a pipeline when you no longer need it. Deleting a pipeline is permanent. To keep a back up, export the pipeline before you delete it.
  1. From the Home page, click the More icon for the pipeline. Or, in the Pipeline library, click the Actions icon for the pipeline.
  2. Click Delete.
    A confirmation window appears.
  3. To delete the pipeline, click Yes.
