Solr

The Solr destination writes data to a Solr node or cluster.

When you configure the Solr destination, you configure connection information for the node or cluster. You also define the indexing mode and map incoming fields to Solr fields. Mapped fields should have compatible data types.

You can configure the destination to index records individually or in batches.

Index Mode

The index mode determines how the Solr destination indexes records when uploading to Solr. Index mode also determines how the destination handles errors.

You can use the following index modes:
Record
The destination indexes one record at a time, and then passes the record to Solr.
If an error occurs, the destination passes the record to the stage for error handling.
Batch
The destination indexes a batch of records at one time, and then passes the batch to Solr.
If an error occurs, the destination rolls back any records that were indexed and passes the entire batch to the stage for error handling.

Configuring a Solr Destination

Configure a Solr destination to write data to an Solr node or cluster.
  1. In the Properties panel, on the General tab, configure the following properties:
    General Property Description
    Name Stage name.
    Description Optional description.
    Required Fields Fields that must include data to be passed to the stage.
    Tip: You might include fields that the stage uses.
    On Record Error Error record handling for the stage:
    • Discard - Discards the record.
    • Send to Error - Sends the record to the pipeline for error handling.
    • Stop Pipeline - Stops the pipeline.
  2. On the Solr tab, configure the following properties:
    Solr Property Description
    Instance Type Solr instance type to write to:
    • Single Node - Writes to single Solr node.
    • SolrCloud - Writes to Solr cluster.
    Solr URI URI for the node.
    ZooKeeper Connection String Connection string for the ZooKeeper associated with the cluster.
    Record Indexing Mode Determines how records are indexed.
    Fields Map fields from the record to Solr fields. Use the Add icon to create additional field mappings.
