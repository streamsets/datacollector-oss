Destinations

A destination stage represents the target system for a pipeline. You can use one or more destinations in a pipeline.

You can use the following types of destinations in a pipeline:
  • Hadoop FS - Writes data to a file system on Hadoop.
  • Kafka Producer - Writes data to a Kakfa cluster.
  • SDC Record Files - Writes Data Collector records to a local file system.
  • Trash - Removes data from the pipeline.
