Shutting Down the Data Collector

Shut down and restart the Data Collector to apply changes to the Data Collector configuration file, environment configuration file, or user logins.
To use the command line to shut down the Data Collector, use the Data Collector process ID in the following command:
kill -3 <process ID>

To use the Data Collector console for shut down:

  1. Click Administration > Shut Down.
  2. When a confirmation dialog box appears, click Yes.
